# Bobo's Monster Band 🎵🥁

A fun musical memory game for Bobo! Watch the cute monsters play their instruments, then repeat the tune. Can you remember the pattern?

## Play the Game

**🎮 [Play Bobo's Monster Band Now!](https://n2l3s4.github.io/hermes-game-2026-04-24/)**

## How to Play

1. Press **Start Game!**
2. Watch carefully as the monsters light up and play sounds
3. Click the monsters in the same order to repeat the tune
4. Complete all 5 levels to win!
5. You have 3 hearts — if you make a mistake, you lose one

## Features

- 🎵 Musical memory gameplay with cute monsters
- 🎨 Colorful, kid-friendly design
- 🔊 Fun sound effects using Web Audio API
- 📱 Works on phones, tablets, and computers
- ⭐ 5 levels that get progressively harder
